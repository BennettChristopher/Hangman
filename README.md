# Hangman
Data Structures - Mini Project

--introduction--

    Hangman is a word guessing game for two or more players. One player thinks of a word, phrase or sentence 
    and the other(s) tries to guess it by suggesting letters within a certain number of guesses.
      -the code will generate a word from the dictionary which has to be guessed by the player.
      -to provide a simple command-line interface that implements the logic of game.
